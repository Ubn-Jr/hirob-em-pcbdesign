
# Hi-Rob Test Board PCB Design
## Required Components
- Esp32 DevkitS V1.1
- Mx1508 Motor Driver 
- N20 Encoder Gear Motor 

## Pin Diagram
- ### Motor Left 
  - Encoder Pin (C1) --> GPIO 14

  - Encoder Pin (C2) --> GPIO 12

- ### Motor Right
  - Encoder Pin (C1) --> GPIO 27

  - Encoder Pin (C2) --> GPIO 26

- ### Motor Driver 
  - INT 1 --> GPIO 35

  - INT 2 --> GPIO 34

  - INT 3 --> GPIO 23

  - INT 4 --> GPIO 22
